=============================================================
Account Payment Term: Disallow deleting terms that are in use
=============================================================

Don't allow deleting payment terms, if they are in use for:

 * Partners
 * Sale orders
 * Purchase orders
 * Account moves

Configuration
=============
* None needed

Usage
=====
* Try to delete a payment term that is in use, and an error message is shown.
